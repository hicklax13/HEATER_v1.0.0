## 2. HEATER Current Feature Summary

HEATER is a deeply analytical fantasy baseball tool with 190+ features across 50+ source files, backed by 1,590 passing tests. Key areas where HEATER already matches or exceeds FantasyPros:

### Where HEATER Already Matches FantasyPros
- Mock draft simulator with AI opponents
- Trade analyzer (far more sophisticated: 6-phase pipeline vs expert-opinion-based)
- Trade finder (cosine dissimilarity + acceptance probability)
- Trade value chart (0-100 scale with G-Score adjustment)
- Waiver wire assistant (LP-verified, 7-stage pipeline)
- Start/sit advisor (3-layer decision model)
- Lineup optimizer (PuLP LP solver, 11-module enhanced pipeline)
- Player comparison tool
- Free agent rankings
- Matchup planner (weekly matchup ratings with color tiers)
- Two-start pitcher planner
- ADP data (3 sources: FanGraphs, FantasyPros ECR, NFBC)
- Park factors (30 teams)
- Injury model and health badges
- Depth chart role classification
- Yahoo league sync
- Post-draft grading
- Category balance analysis
- Probable pitcher tracking (via two-start planner)

### Where HEATER Significantly Exceeds FantasyPros
- **Monte Carlo simulation** (100-10K sims with variance reduction)
- **Bayesian Model Averaging** (posterior-weighted projection blending)
- **Gaussian copula** (correlated stat sampling)
- **Kalman filter** (true talent estimation)
- **BOCPD regime detection** (changepoint detection + HMM)
- **Game theory** (Vickrey auction, adverse selection, Bellman DP)
- **Signal decay** (sport-specific half-lives for Statcast features)
- **Stochastic MIP** (scenario-based lineup optimization)
- **7 projection systems** (Steamer, ZiPS, Depth Charts, ATC, THE BAT, THE BAT X, Marcel)
- **Concentration risk** (HHI scoring for roster diversification)
- **LP-constrained trade evaluation** (only starters' stats count, bench excluded)
- **Roster cap enforcement** (forced drop/FA pickup modeling in uneven trades)
- **CVaR/VaR risk metrics** for trade evaluation
- **Convergence diagnostics** (ESS, split-R-hat)
- **News sentiment scoring** (keyword-based, ready for API integration)
- **Draft backtesting** (RMSE, rank correlation, value capture rate)

## 3. Gap Analysis: Features NOT in HEATER

### TIER 1 — High Impact, High Feasibility

| # | FantasyPros Feature | Gap Description | Complexity |
|---|---------------------|-----------------|------------|
| 1 | **Expert Consensus Rankings (ECR)** | HEATER uses projection systems but does not aggregate human expert rankings from 50+ experts. ECR is FantasyPros' core differentiator — consensus of human expertise rather than purely algorithmic. | Medium |
| 2 | **Multi-Platform League Sync** | HEATER only syncs with Yahoo. FantasyPros supports Yahoo, ESPN, CBS, Fantrax, FleaFlicker, RTSports, Sleeper, NFC. ESPN alone represents ~40% of the fantasy baseball market. | High |
| 3 | **Auction/Salary Cap Draft Mode** | HEATER is snake-draft-only. No auction draft simulator, dollar values, salary cap management, or auction strategy tools. Auction leagues are ~20-30% of the market. | High |
| 4 | **Live Draft Sync** | HEATER cannot connect to and sync with a live draft happening on Yahoo/ESPN. FantasyPros detects picks in real-time and auto-crosses off players. | High |
| 5 | **Multiple Scoring Format Support** | HEATER is hardcoded for H2H Categories (6 hitting + 6 pitching). No Roto, H2H Points, or custom points league support. | High |
| 6 | **Cheat Sheet Creator/Export** | No printable, customizable draft cheat sheet. Cannot export rankings to PDF or print format. | Low |
| 7 | **ADP Trend Tracking** | HEATER stores current ADP but does not track ADP movement over time. FantasyPros shows rising/falling ADP with charts. | Medium |

## 4. Priority Recommendations

### Must-Have (Build Next)

These features would close the biggest gaps with FantasyPros and serve the broadest user base:

1. **Cheat Sheet Creator/Export** (Low complexity, High value)
   - Printable/PDF draft cheat sheet from current rankings
   - Customizable by position, tier breaks, user notes
   - Fastest win for user-facing value

2. **ADP Trend Tracking** (Medium complexity, High value)
   - Store daily ADP snapshots in SQLite
   - Visualize rising/falling players over time
   - Add to draft page as supplementary data

3. **Buy Low / Sell High Classification** (Low complexity, Medium value)
   - Compare recent performance vs ROS projections
   - Flag regression candidates (buy low) and over-performers (sell high)
   - Add to trade finder and free agent pages

4. **League Analyzer Page** (Medium complexity, High value)
   - Power rankings across all 12 teams
   - Per-team strengths/weaknesses by category
   - Competitive landscape visualization
   - Leverages existing standings and roster data

5. **Projected Final Standings** (Medium complexity, Medium value)
   - Monte Carlo simulation of remaining season
   - Playoff probability calculator
   - Surface the existing `estimate_playoff_probability()` as a standalone tool

### Should-Have (Build Later)

6. **Auction Draft Mode** — Opens HEATER to ~25% of the market currently locked out
7. **Multi-Scoring Format Support** — Roto and Points league support would significantly broaden appeal
8. **ESPN League Sync** — ESPN represents ~40% of the fantasy baseball market
9. **Weekly Rankings Page** — Distinct from ROS, focused on this week's matchups
10. **Dynasty/Keeper Rankings** — Long-term value rankings for multi-year leagues

### Nice-to-Have (Consider for Future)

11. **Player Notes/Tags** — Personal annotations and watchlist
12. **Multi-Player Comparison** — Compare 3+ players side by side
13. **Sleeper/Bust Designations** — Pre-season tags with reasoning
14. **Trending Players** — Most added/dropped tracking
15. **Strength of Schedule Page** — Dedicated SoS visualization
16. **Expert Accuracy Tracking** — Backtest projection systems against actuals (backtesting module exists, needs UI)
17. **Probable Pitchers Page** — Dedicated daily SP schedule view

### Different Business Model (Not Recommended)

These are core to FantasyPros' business but don't align with HEATER's architecture:

18. **DFS Tools** — Entirely different product category (daily salary-based vs season-long)
19. **Native Mobile Apps** — Would require React Native/Flutter rewrite; Streamlit is mobile-responsive
20. **Chrome Extension** — Requires separate codebase and Chrome Web Store distribution
21. **Content Ecosystem** — Articles/podcasts require editorial team, not a software feature
22. **Coach AI** — Could be built with LLM integration but is a separate product initiative
23. **Multi-League Management** — Requires significant architectural changes to support N leagues
24. **Live Draft Sync** — Requires real-time websocket integration with Yahoo/ESPN draft rooms

### Key Insight

HEATER dramatically outperforms FantasyPros in **analytical depth** — Monte Carlo simulation, Bayesian methods, game theory, signal processing, and optimization are leagues beyond what FantasyPros offers. FantasyPros' advantage is in **breadth and accessibility** — expert aggregation, multi-platform support, mobile apps, content ecosystem, and DFS tools. The recommended strategy is to close the highest-impact accessibility gaps (cheat sheets, ADP trends, league analyzer) while maintaining HEATER's analytical superiority as its core differentiator.

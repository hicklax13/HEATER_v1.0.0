How it works: Each probability is computed using Bayesian-updated Normal approximation with category correlations (Gaussian copula). Your team's projected weekly production is estimated from ROS projections for active starters, weighted by PA (hitting) or IP (pitching). Variance shrinks as the season progresses (more data = more confidence). Correlated categories (HR/R/RBI, ERA/WHIP) are modeled together.

Color coding: Green (70%+) = strong advantage   Blue (55-70%) = slight edge   Orange (45-55%) = toss-up   Red (<45%) = disadvantage

For past weeks: Shows actual results (W/L) instead of probabilities. For future weeks: uses projected rosters and ROS projections.
Key changes from current Matchup Planner:
• Week navigator (left/right arrows) defaults to current week, can browse all weeks 1-24
• New "Category Probabilities" tab — the headline feature: per-category win probability bars
• Past weeks show actual results (W/L per category) instead of probabilities
• Future weeks show projected probabilities using ROS projections + opponent roster
• Existing tabs preserved — "Player Matchups" (renamed from Summary) and "Per-Game Detail" stay
• Context panel — week navigator, overall win probability, projected category score, filters
• Reco banner — overall win %, best/worst categories, focus areas
